## 서술형 수행평가 연습 문제
**전시회 입장 프로그램**

다음 조건에 맞는 전시회 입장 프로그램을 작성하시오.

* 프로그램 시작 파일 이름은 main.py로 한다.
* __main__ 값을 이용하여 프로그램의 시작점을 지정한다.
* 프로그램 실행 후, 관람객의 이름, 나이, 관람할 전시회명을 입력받아 입장 가능 여부를 출력한다.
* 입력과 출력은 반복할 수 있어야 한다.
* 첫 번째 입력값으로 영문자 X 또는 x를 입력하면 프로그램이 종료된다.
* 전시회명, 입장료, 제한 연령은 2차원 리스트 exhibit에 저장한다.
* 입력한 값은 딕셔너리 client에 저장하고, 키 값은 이름으로 한다.
* exhibit_info() 함수를 작성하여 전시회명에 따라 입장료와 제한 연령을 찾도록 한다.
* 평가 요소에 해당하는 부분에는 정확한 주석문을 작성한다.
